# 김도희 님에 대한 정보

**구조체(포인터) 사용**

[2020.08.02.일]

```c
#include <stdio.h>
#include <string.h>
#include <stdlib.h> //melloc, free가 담겨 있는 헤더파일

struct Senior{
	char name[5];
	int age;
	char nickname[100];
};  

int main(){
	struct Senior *p1 = malloc(sizeof(struct Senior));
	strcpy(p1->name, "김도희");
	p1->age = 18;
	strcpy(p1->nickname, "멋지고 예쁘고 착한 선배님");
	printf("%s\n", p1->name);
	printf("%d\n", p1->age);
	printf("%s\n", p1->nickname);
	
	free(p1);
	 
	return 0; 
}
```

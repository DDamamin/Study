**[기초 - 산술 연산] 정수 1개 입력 받아 부호 바꿔 출력하기**

```c
#include <stdio.h>

int main(){
	int a; // 정수니까 int를 쓴다. 
	scanf("%d", &a); // int이니까 %d를 쓰고 a를 입력해준다.
	printf("%d", -a); // a를 출력할 때 부호를 바꿔 출력해야 해서 -를 붙인다.
    
    return 0; // 이 리턴은 꼭 써주기
}
```

2진수   - 두개의 기호 ( 0 , 1 )를 이용해 데이터를 표현하는 방식

10진수  - 이와 유사하게 열 개의 기호( 0~9 ) 를 이용해 데아터를 표현하는 방식

16진수  - 열 여섯 개의 기호  ( ( 0 ~ 9 ) + ( A~ F ) ) 를 이용해 데이터를 표현하는 방식

 

 

16 진수를 사용하는 이유

 

2진수로 데이터를 표현하면 그 길이가 길어져 표현하기에도 , 한 눈에 파악하기도 어렵다.

그러나 16 진수를 사용한다면 2진수를 간단히 표현 , 파악 할 수 있다.

( 2진수 4개가 16진수 1개로 표현 가능하기 때문 )
 

 

 
 

'비트'는 컴퓨터 데이터의 최소단위로서 2진수 값 하나를 저장할 수 있는 메모리의 크기를 뜻하는 단위이다.

 


비트 8개를 묶으면 바이트가 된다.

 


 

정수를 표현하는데 있어 가장 왼쪽에 존재하는 비트는 부호를 표현하는데 사용된다.

그리고 나머지 비트들은 데이터의 크기를 나타낸다.

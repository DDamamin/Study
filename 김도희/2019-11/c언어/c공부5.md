 

관계 연산자 - 대소와 동등의 관계를 따지는 연산자 (  = 비교 연산자 )

 

- 위의 관계 연산자들은 조건을 만족하면 1 ( 참 ) 을 , 만족하지 않으면 0 ( 거짓 ) 을 반환한다. 


 

논리 연산자 - 논리곱( AND /  && ) , 논리합 ( OR  /  || ) , 논리부정 ( NOT  /  !  ) 을 표현하는 연산자
 

 

c언어는 0이 아닌 값을 모두 참으로 간주한다.

즉 , 거짓을 의미하는 숫자는 유일하게 0 하나고 , 0이 아닌 모든 숫자는 '참'으로 인식된다. 다만 그중에서 '참'을  의미할때 주로 1이 사용되고 결과가 참이 반환되어야 할때 1이 반환되는 것이다.

 
 

콤마 연산자 - 둘 이상의 변수를 동시에 선언하거나, 둘 이상의 문장을 한 행에 삽입할 때 사용되는 연산자

 

- 다른 연산자들과는 달리 연산의 결과가 아닌 구분을목적으로 주로 사용된다.

 
 

연산자 우선 순위 - 우선 순위가 높게 정의되어 있는 수식을 먼저 계산하는 것

 
 

 

결합 방향 - 우선 순위가 동일한 두 연산자가 하나의 수식에 존재하는 경우 , 어떤 순서대로 연산하느냐를 결정해 놓은것

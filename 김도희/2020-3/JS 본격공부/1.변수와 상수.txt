1. 변수와 상수

변수와 상수는 특정한 이름에 특정값을 넣을 때 사용한다. 
예 ) let age = 18;
- 위와 같은 경우 앞으로 우리가 age를 조회할 겨우 age는 18을 가르키게 된다.

console.log(age)를  할 시에는 웹 콘솔창에 18이 출력되는 것을 볼 수 있다.




2. 변수와 상수의 차이

1)변수란 변할 수 있는 값을 말하며 한 번 선언한 뒤 값을 바꿀 수 있다. (주의할 점으로는 한 번 변수를 선언할 경우 같은 이름으로 선언할 수 없다.)
예 ) let num = 1;  console.log(num);  -> 값으로 1이 출력 
          num = 2;  console.log(num);  -> 값으로 2가 출력
      let num = 3;  -> 같은 이름으로 선언했으므로 오류 발생(단 블록 범위 내에서는 사용가능 + 나중에 내용 추가)


2) 상수란 한번 선언한 뒤 바뀌지 않는 값을 말한다. 즉 고정적인 값을 말한다.
예 ) const var = 1;
- 위와 같은 경우 앞으로 우리가 var를 조회할 경우 var는 1을 가르키게 된다.
- 단 값을 변경 시 오류가 뜬다. 또한 let과 같이 같은 이름으로 선언 시 오류가 발생한다.
예 ) const num = 1;
              num = 2;     ->   오류 발생
      const num = 3;     ->   오류 발생



3. 과거에 사용하던 변수 선언 키워드 var
- 더 이상 사용하지 않는 키워드이다. 
- let과 다른 점으로는 똑같은 이름으로 여러번 선언이 가능하다.(구형 브라우저에서 사용됨)
예 ) var num = 1;
      var num = 2;  -> 다른 키워들과는 다르게 오류가 발생하지 않음

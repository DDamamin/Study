
상수 - 그 이름이 의미하듯이 변경이 불가능한 데이터를 뜻함

 

리터럴 상수 - 변수와는 달리 이름이 없는 상수 

( 예로 num = 10 + 2 ; 에서 메모리 공간에 이름이 있는 num( 변수 ㅔ)과는 다르게 30 ,40 (리터럴 상수)은 이름이 없음 )


 

심볼릭 상수 - 변수와 마찬가지로 이름을 지니는 상수

( 예로 상수로 선언해주는 const 상수가 있다. ) 

 

 

자료형의 반환 - 데이터의 표현방식을 바꾸는 것
 

 

자동 형 변환 ( 묵시적 ) - 자동으로 발생하는 형변환

 

- 대입연산자의 왼쪽과 오른쪽의 연산자가 자료형이 다르면 왼쪽 연산자를 기준으로 형변환이 자동으로 형이 변환됨 

- 정수의 승격 ( 작은 형태의 형이 속도가 빠른 다른 형태의 형으로 변환되어 연산된는것 ) 으로도 발생



 

강제 형 변환 ( 명시적 ) - 프로그래머가 형 변환을 명시해서 강제로 변환이 일어나게 하는것

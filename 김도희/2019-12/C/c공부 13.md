switch문의 기본 형태
 

 

switch( 전달되는 인자의 정보 )

{

     case 내가 필요한 조건 :     <-  조건은 계속 추가가능

     break;

 

     case 내가 필요한 조건 : 

     break;

}

 

 

- switch문에서 break문은 선택적으로 입력해야 한다. ( 경우에 따라 유용할때도 , 하지 않을때도 있다. ) 

 

- 또 분기가 많아질 경우 가급적 switch문을 사용하는 것이 좋다. ( 실제로 많은 프로그래머들이 선호함 )

 

 

 

 

goto문은 프로그램의 자연스러운 흐름을 방해함으로 사용하지 않는 것이 좋다.

 

- c 언어와 같은 절차지향 프로그램에서 프로그램의 흐름을 복잡하게 하거나 방해하는 것은 아주 큰 단점이다. 

 

 

 

 

 

goto문 형태          //  아까도 말했지만 사용방법만 알려줄뿐 사용은 추천하지 않는다.

 

int main( )

{

  rabbit:

 

     ......

 

  goto rabbit :              // rabbit이라는 위치로 이동

}

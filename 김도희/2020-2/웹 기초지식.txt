<!doctype>
-문서 유형을 지정하는 선언문

웹 문서의 시작을 알리는 <html>태그보다 먼저 사용되는 '문서 유형'을 지정하는 태그
( 이때 지정하는 문서 유형이란 웹 브라우저에게 이제부터 처리할 문서가 HTML 문서이며 어떤 유형을
사용했으니 그 버전에 맞는 방법으로 해석하라는 의미를 담고 있다. )

과거 HTML4에서는 문서 유형이 여러가지인데다가 문서 유형을 지정하는 소스도 길어서 잘못 사용하거나 
아예 문서 유형을 사용하지 않는 경우들도 있었는데 HTML5부터는 단 한줄로 문서 유형을 선언한다.
그게 바로
<!doctype html>이다.

-------------------------------------------------------------------------------------------------------------------------

<html> 태그
- 웹 문서의 시작을 알리는 태그

문서 유형을 선언한 후 실제 문서 정보와 내용이 시작되고 끝나는 것을 표시하는 태그
(따라서 <html> 태그는 시작을 , </html> 태그는 끝을 의미한다. )

<html> 태그에서는 lang 속성을 사용해 문서에서 사용할 언어를 지정할 수 있다.
(예) <html lang="ko"> <- 사용 언어가 한국어라는 의미

언어 표기는 ko처럼 두 자리를 사용할 수 있고 en-US처럼 네 자리를 사용할 수도 있다.
아래는 코드 참고!!

de : 독일어  en : 영어  fr : 프랑스어   ja : 일본어   ko : 한국어   zh: 중국어

사용언어를 명시하는 이유
- 검색 사이트에서 특정 언어로 제한해 검색할 때 그 대상이 될 수 있기 때문
- 화면 낭독기가 웹 문서를 소리내어 읽을 때  명시한 언어에 맞추어 억양, 목소리를 다르게 들려 줄 수 있음

<html> , </html> 사이에는 문서 정보를 지정하는 <head> 태그와 실제 화면에 보이는 <body>태그가 포함


---------------------------------------------------------------------------------------------------------------------------------

<head> 태그
- 브라우저에게 정보를 주는 태그

웹 브라우저 화면에는 보이지 않으나 웹 브라우저가 알아야 할 정보들은 모두 <head> 부분에 입력
( 문서에 사용할 외부 파일들도 이곳에서 링크한다. )

<head>와</head> 태그 사이에 문서 정보를 표시하기 위해 사용할 수 있는 주요 태그들은 다음과 같다.

<title> 태그 , <meta> 태그

이 외에도 <style>태그와 <link>태그 등이 사용된다.

------------------------------------------------------------------------------------------------------------------------------------

<title> 태그
- 문서의 제목을 나타내는 태그

<title> 태그에서 지정하는 내용은 거의 모든 웹 브라우저의 제목 표시줄에 표시되고 해당 페이지를 방문하는 
방문자나 검색 엔진은 제목 표시줄의 제목을 보고 페이지 전체의 내용을 추측한다.

형태
<title> 문서제목 </title>
- 즐겨찾기로 추가할때 <title<태그 안의 내용으로 추가된다. ( 페이지 창 위에 표시되는 이름이다. )

------------------------------------------------------------------------------------------------------------------------------------

<meta> 태그
- 인코딩 및 문서 키워드, 요약 정보

<meta> 태그는 화면에 글자를 표시하는 문자 인코딩 방법을 비롯해 웹 문서 키워드 등을 지정할 수 있다.
또한 웹 문서 정보를 검색 엔진에게 전달해 주는 데 매우 유용하게 사용된다.

<meta>태그에서 가장 중요한 것은 문자 인코딩을 지정하는 것
( 웹 서버는 영어를 기본으로 해서 웹 문서에서 한글을 표시하려면 한글을 화면에 나타내기 위한 문자 세트를 지정해야
하는데 이걸 { 문자 인코딩 } 이라 한다. )

HTML5에서는 한글과 영문을 비롯해 모든 언어를 표시할 수 있는 utf-8 방식을 사용한다.
<meta charset = "utf-8">

또한 <meta>태그를 이용해 문서에 관한 간단한 설명을 지정할 수 있는데 이렇게 지정한 설명은 검색 결과에
사이트 설명으로 표시된다.
<meta name="이름" content="사이트에 대한 설명">


-----------------------------------------------------------------------------------------------------------------------------------------

<body> 태그

- 실제 브라우저에 표시될 내용

<body> 와 </body> 태그 사이에는 실제 브라우저에 표시될 내용을 입력한다.

우리가 흔히 웹을 시작할때 배우는 태그들의 대부분이 <body> 태그 내에 사용되는 태그들이다.

예 ) <h1> , <br> , <p> , <img> ......등

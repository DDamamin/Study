# 플랫폼 별

## PC

- 이 pc로 출석체크하기
  PC버전에서 태블릿 버전으로 바꿈

#### 로그인

- 이메일
- 비밀번호

#### 오늘의 알림

- 달력
- 오늘 방과후 활동 표시
- 사전 결속 신청자 표시

#### 감독 교체
- 날짜 선택
	- 달력
- 선택한 날짜에 감독 선생님 표시
- 감독 선생님 바꾸기
- 감독 선생님 저장

#### 학생변동 사항
- 학생 선택
	- 학년 선택
	- 반 선택
	- 번호 이름 선택
- 8, 9, 10 교시 출결 사항 입력
- 비고
- 저장



## Tablet

#### 시작
- 시작
#### 메인페이지
- 오늘의 감독 교사 표시
#### 동아리
- 동아리 선택
- 출석 체크
#### 자율학습
- 반 선택
- 출석 체크
#### 방과후
- 자습하는 사람들 출석 체크

# 기능 별
- 달력
  각 날짜별 감독 선생님 표기 (날짜별 3명 이하)
- 사전 결속 신청자
  방과후 시간 전, 미리 외출이나 현장체험학습을 가거나 학교의 허락을 맡은 학생
- 출결 사항
  학생의 출결을 관리. 3교시와 5가지 상태, 비고를 가진다.
  상태는 기본적으로 출석 상태를 가진다.
  교시 : 8, 9, 10교시
  상태 : 출석, 현체, 외출, 지각, 공결
  비고 : 결석 사유등을 입력

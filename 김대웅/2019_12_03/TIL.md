# 오늘 공부

- 해시 테이블 : 연관배열 구조를 이용하여 키에 결과 값을 저장하는 자료구조

- 연관배열 구조 : 키 1개와 값 1개가 1:1로 연관되어 있는 자료구조. 키를 이용하여 값을 도출할 수 있다.

- 용어

  1. 키(key) : 고유한 값이며, 해시 함수의 input이 된다. 다양한 길이의 값이 될 수 있다.

  2. 해시 함수 : 키를 해시로 바꿔주는 역할을 한다. 다양한 길이를 가지고 있는 키를 일정한 길이를 가지는 해시로 변경하여 저장소를 효율적으로 운영할 수 있도록 도와준다.

  3. 해시 : 해시 함수의 결과물이며, 저장소에서 값과 매칭되어 저장된다.

  4. 값 : 저장소에 최종적으로 저장되는 값으로 키와 매칭되어 저장, 삭제, 검색, 접근이 가능해야 한다.

  5. 해시 충돌 : 서로 다른 입력 값에 대해 동일한 해시 값, 즉 해시 테이블 내의 동일한 주소를 반환하는 것.

     

- 해시 충돌 해결방법
  - 체이닝 : 충돌이 발생하면 각 데이터를 해당 주소에 있는 링크드 리스트에 삽입하여 문제를 해결하는 방법
  - 개방 주소법 : 충돌이 일어나면 해시 테이블 내의 새로운 주소를 탐사하여 충돌된 데이터를 입력하는 방식
